# Demo

Coloca aquí el video del proyecto mostrando las 3 rutas funcionando:

1. `/` — Home con hero, destacados y botón aleatorio
2. `/champions` — Listado con búsqueda y filtro por rol
3. `/champions/:id` — Detalle del campeón (lore, stats, habilidades, skins)

Formato sugerido: `demo.mp4` o `demo.mov`.
